Basic CRUD with Rocket

This is what I went through while creating my first web app in Rust.

cargo init kolektilo

Add a route for static files

use rocket_contrib::serve::StaticFiles;

[ ... ]

fn main() {
    rocket::ignite()
        .attach(Template::fairing())
        .mount("/static", StaticFiles::from(concat!(env!("CARGO_MANIFEST_DIR"), "/static")))
        .mount("/", routes![
            ...
        ])
    .launch();
}

This will take any file within the static directory and serve it from http://localhost:8000/static

Installing Foundation in static

I want to use Foundation for styling. There are a lot of ways to installing it. One way way is to download a customized version of the complete bundle of foundation from itâ€™s download page. Another way would be to use a package manager like npm or bower, a task runner like grunt and install a sass compiler to create an optimized version of css and js files. But the easiest way I found is to install the foundation-cli to create all the boilerplate for me.

First step is to make sure you have an updated version of npm

npm install -g npm

Then install/update the foundation-cli package:

npm install -g foundation-cli

Now itâ€™s time to generate the boilerplate into the staticdirectory.
